# Rem to Pixel Converter | Fridge Combo #78

A high-contrast, zero-friction unit converter designed for modern web developers and designers. Stop wasting mental energy on 16-bar calculations—get your CSS units instantly.

## Core Philosophy
- **1 Tool / 1 Action**: Real-time, bidirectional conversion between `rem` and `px`.
- **High Visibility**: Optimized contrast and large typography for professional use in any environment (even for tired eyes after a long shift).
- **Zero-Waste Workflow**: One-tap "Clear All" and "Tap to Copy" features to keep your coding flow uninterrupted.

## Key Features
- **Bidirectional Sync**: Type in either field; the other updates instantly.
- **Adjustable Base Size**: Change the root font size (default 16px) with a single tap to match any project's requirements.
- **Quick Scale Table**: A reference list of common sizes (8px, 12px, 24px, etc.) that updates dynamically.
- **Smart Input**: Optimized for mobile and desktop, supporting decimals and negative values for advanced CSS layouts.

## How to Use
1. **Set Base**: Adjust the "Base Size" if your project doesn't use the standard 16px.
2. **Convert**: Type your value into the REM or PX box.
3. **Copy**: Tap any value in the "Quick Scale" table to copy the `rem` value directly to your clipboard.
4. **Clear**: Use the `×` button to reset both fields and start over.
